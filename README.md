# QR Code Maker

## How to run

### Requirements

- python 3.10 or greater
- shell

### How to use

Go into the main.py file and replace the value of text (line 17) by any url you want then run the file. The generated QRCode will appear in the folder.
```commandline
pip install -r requirements.txt
python3 main.py
```
